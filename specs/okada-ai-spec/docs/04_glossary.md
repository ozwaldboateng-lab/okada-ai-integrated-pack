# 用語集

## 岡田理論側の共通語彙
### dominant route
その状況で主経路として機能している経路・説明・方策。

### retained history / retained load
過去の履歴・残留・惰性・負債として現在判断に残っているもの。

### competitor pressure
主経路に対抗する競合経路・競合仮説・競合シグナルの圧力。

### first visible handle
崩れ・汚染・分岐の最初の可視入口。

### regime
- clean
- mixed
- contaminated

### type
- Type I: 主経路の局所破綻
- Type II: 競合枝の侵入
- Type III: 経路構造の喪失・複雑化

## AI実装側の共通語彙
### trust window
通常運用を継続してよい範囲。

### contamination
主経路が保てなくなり、競合や残留が支配的になった状態。

### abstain
回答や自動実行を控えること。

### handoff
人間または別経路へ引き渡すこと。

### bounded hybrid
限定的な併用。

### branchwise hybrid
分岐単位で異なる経路を混在させること。

# Integrated Stack Matrix

## Directly started by compose in this pack
- Okada Kernel Service
- LiteLLM Proxy
- Open WebUI

## Linked externally / manually
- Dify
- LangGraph runtime

## Why
The compose path in this pack focuses on the shortest honest path to an integrated operator-visible stack.
Dify and LangGraph are still fully represented in examples, runbooks, and runtime glue, but not forced into a misleading single-file compose claim here.

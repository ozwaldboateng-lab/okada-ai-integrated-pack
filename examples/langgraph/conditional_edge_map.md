# Conditional Edge Map

## Before-tool governance
- `continue` -> `tool`
- `constrained_continue` -> `tool`
- `replan` -> `replan`
- `human_handoff` -> interrupt, then reviewer decides
- `abort` -> `final`

## After-tool governance
- `continue` -> `final`
- `constrained_continue` -> `final`
- `replan` -> `replan`
- `human_handoff` -> interrupt, then reviewer decides
- `abort` -> `final`
